MAIN MISSION

Litteray was created for NFT influencers to make the communication between the Artist and the Holder better.

THE CREATORS

LItteray was created by the main programmer of JoelTeam as the diploma project.
The theme of project: Programming
Teacher: Dilara Abzhanova

NAME MEANING

Title “Litteray” was inspired by the Greek word “Litterae” which means “Letter” or “Dispatch” - the fast letter.
The code name of the App: “WhatsApp 2”

LITTERAY ON IOS

Litteray is a diploma project of Java courses. The opportunity of appearing Litteray on iOS devices is smooth.

OPEN-SOURCE

Litteray is open-source messenger. The code of Litteray is available on GitHub page of JolyTh3Kidd

DIFFERENCE

Why do I need Litteray if there’s many apps like this?
The main aim of Litteray is to help NFT Artists and Holders communicate together, how we mentioned above. But the other messengers as Telegram, WhatsApp, etc. can’t provide such a level of comfort of communication for NFT-influencers as Litteray.
